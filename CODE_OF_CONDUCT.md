# Simple Code of Conduct (by retro-person)
## General
This code of conduct's goal is to...
* Create an open and friendly environment.
* Maintain a happy attitude. : )
* Providing a harrassment-free place.
* Maintain inlcusive language and constructive criticism.
## Enforcement

Doing the stuff above.
### What's Not Cool
Doing the opposite of the stuff above.
### If someone does something not cool...
* Corrective action wil be taken in response, e. g. deleting a mean comment or pull request.
